# Higher Education Analytics Dashboard (Power BI)

Interactive Power BI dashboard analyzing 609 colleges, 12,614 courses, and ~2M students to uncover regional distribution, management patterns, and institutional capacity insights.

---

## Key Metrics

- Total Colleges: 609  
- Total Courses: 12,614  
- Total Students: ~2,000,000  
- Average Students per College: 3,440  

---

## Dashboard Pages

| Page | Description |
|-----|-------------|
| Main Dashboard | Overview of colleges, students, courses, and distribution |
| Geographical Overview | Regional and state-wise distribution analysis |
| Management Overview | Private vs Government vs Autonomous comparison |
| Capacity Analysis | Relationship between student strength and course count |
